# mDL แบบ Offline — Live Demo

> ## 🔴 [▶ เปิด demo (กดปุ่ม/เล่นแอนิเมชันได้จริง)](https://dga-sd.github.io/Claude-SKILL/flow-sim/example/mDL-offline-demo/)

ตัวอย่างผลลัพธ์จริงของ skill [`flow-sim`](../../) — flow จำลองการแสดงใบขับขี่ดิจิทัล (mDL) แบบ offline ตาม **ISO/IEC 18013-5**

- **แท็บ 1 — เฟสเตรียมล่วงหน้า**: ออก mDL (mdoc + MSO) เข้ากระเป๋า + เครื่องตรวจโหลด Trust List (IACA)
- **แท็บ 2 — เฟส ณ จุดตรวจ (offline 100%)**: QR Device Engagement → BLE → Selective Disclosure → ตรวจครบชุดในเครื่อง โดยไม่แตะอินเทอร์เน็ตเลย
